## 🖼️ Sprite Animation Sets
The character sprite folders (e.g., `Potent Monkeyrog/`) contain frame-by-frame PNG images for animations such as:
- `stand_0.png`, `stand_1.png`, … — idle/standing animation
- `move_0.png`, `move_1.png`, … — walking/moving animation
- `attack1_0.png`, `attack1_1.png`, … — attack animation
- `die1_0.png`, `die1_1.png`, … — death animation
- `hit1_0.png`, … — hit/damage animation
