# 🎓 CSE Summer Camp Course

This folder contains exercises and mini-projects created during a **Computer Science & Engineering (CSE) Summer Camp** course.  
The projects range from basic Python programming exercises to full Pygame games.

---

## 📁 Contents

### 📝 Python Exercises
| File | Description |
|------|-------------|
| `Q1.py` | Input your name & birthday, perform date-based arithmetic calculations |
| `Q2.py` | Generate Pascal's Triangle up to `n` rows |

### 🕹️ Mini-Games
| Folder | Description |
|--------|-------------|
| [pygame小遊戲](./pygame小遊戲/ReadMe.md) | A small Pygame sprite game with collision detection and sound effects |
| [五子棋](./五子棋/ReadMe.md) | Gomoku (Five in a Row) — a two-player board game |

---

## 🚀 Getting Started

Make sure **Python 3.x** and **Pygame** are installed:

```bash
pip install pygame
```

Then navigate into any sub-folder and run its main script.

---

## 🗒️ Notes
- `Q1.py` and `Q2.py` are plain Python scripts (no Pygame required).
- The mini-games require Pygame and should be run from **within their own folder** so that relative asset paths resolve correctly.
