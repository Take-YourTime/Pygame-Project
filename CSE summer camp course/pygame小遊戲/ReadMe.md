# 🕹️ pygame小遊戲 (Small Pygame Game)

A beginner-level Pygame sprite game created as part of the CSE Summer Camp course.  
The game features moving sprites, collision detection, and sound effects.

---

## ✨ Features
- **Sprite-based gameplay** using `pygame.sprite.Group`
- **Collision detection** between player and other sprites
- **Sound effect** played on collision
- **Random colors** for sprites (red, green, blue)
- **60 FPS** game loop

---

## 📁 File Structure
```
pygame小遊戲/
├── pygame_sprite.py   # 🎮 Main game file — run this to start
└── sound/
    └── collision.wav  # Collision sound effect
```

---

## 🚀 How to Run
1. Open this folder in **VS Code** (or any terminal).
2. Make sure **Pygame** is installed:
   ```bash
   pip install pygame
   ```
3. Run the main file:
   ```bash
   python pygame_sprite.py
   ```

---

## 🗒️ Notes
- Window size: **600 × 600**.
